# Lightning App

## Setup
### Install dependencies

``` npm i```

### Add LND Domain
Add LND Domain to `.env`. 
Rename the file `.env.example` to `.env`. 

```$ cd ./src```

```LND_DOMAIN=<DOMAIN>:<RPC-PORT>```

### Run
#### Create the dev server
```make dev```

#### Open the server in the browser
```make open```
